## Offset for overlapping bus routes
[Bus_Routes_GeoJSON.ipynb](https://github.com/iliberts/projects/blob/main/Bus_Routes_GeoJSON.ipynb)
<br>
When making a route map, one of the visual challenges is displaying overlapping route sections. I walk through the process of creating a Leaflet map of bus routes with offset overlapping portions, all without altering the actual routes. App: https://geo.sigulda.lv/marsruti/ iframed: https://sigulda.lv/izglitiba/izglitibas-nodala/skolenu-autobusu-marsruti/
